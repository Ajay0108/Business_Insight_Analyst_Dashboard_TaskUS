📊 Operations & CSAT Performance Diagnostic Dashboard

🎯 Project Overview
An enterprise-grade Power BI diagnostic ecosystem built to analyze and cross-examine operational throughput, process quality, and customer sentiment. Moving away from standard, isolated metrics, this dashboard implements a multi-layered analytical framework to evaluate agent performance across standard efficiency quartiles and shift structures, successfully isolating critical operational trade-offs and data integrity risks.

🛠️ Tech Stack & Skills Showcased
Data Visualization: Power BI Desktop (Advanced custom layouts, matrix grids, and interactive multivariate charts).

Data Analysis & Scripting: SQL / Google Sheets QUERY (Advanced grouping, multi-metric aggregations, and data preparation).

Analytical Modeling: Tri-metric correlation tracking, outlier analysis, and trend forecasting across tenure buckets.

Design System: Structured color-coded conditional formatting (Teal/Purple compliance coding and Alert Red/Pink anomaly highlights) for instant visual discovery.

🔍 Core Analytical Deep-Dives
1. Balanced Scorecard Framework

    -Leverages a synchronized 70/30 weighted distribution (70% CSAT / 30% Error Rate Compliance) to evaluate baseline delivery, protecting       fast handlers from volume bias while ensuring technical execution remains stable.

2. Tri-Metric Correlation & Outlier Analysis

    -Utilizes a multivariate scatter plot mapping CSAT and Quality scores spatially, while scaling marker volumes by Average Handle Time         (AHT).

    -Successfully isolates high-risk behavioral profiles, such as "People Pleaser" anomalies (agents securing maximum customer satisfaction      at the absolute expense of process compliance) and severe execution bottlenecks.

3. Structural Trends & System Anomalies Uncovered

    -The Tenured Quality Slump: Discovered that process error rates do not drop linearly with tenure; instead, they peak at 8.64% within         the 7–9 month phase, indicating clear operational fatigue.

   -The "Impossible CSAT" Glitch: Isolated database synchronization duplicates where individual rows recorded mathematical impossibilities      (CSAT percentages exceeding 100% due to good surveys outnumbering total survey counts).

   -Shift Fatigue Tracking: Proved that technical compliance remains stable 24/7 (flat error rates across AM, PM, and MID shifts), while        customer sentiment drops steadily by 3.88% during overnight shifts, pinpointing a clear soft-skill training opportunity.
